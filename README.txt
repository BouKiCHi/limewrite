LimeWrite ver 1.01

*このツールは？

　LPCマイコン書き込み補助ツールです。
mbedなどで生成されるbinファイルをこのツールで書き込む事ができます。

*既知の不具合
　ファイル選択後に書き込みができない（だいぶ致命的）

→HSPの不具合だと思われます。(もしくはWindows 10)
　ファイル選択の代わりにドラッグアンドドロップを行うか、
LimeWriteをリスタートさせてください。
